# UrbanNest Entebbe

UrbanNest is a specialized property rental listing platform focused on the Entebbe, Uganda market. It helps users find affordable rental properties including single rooms, studio apartments, and small family houses.

## Features
- **Curated Listings**: Focused on smaller, affordable rentals (Single rooms, 1-3 roomed apartments).
- **Realistic Pricing**: All prices are adjusted to match the current Entebbe rental market.
- **Enquiry Management**: Potential tenants can send enquiries directly to landlords.
- **Landlord Dashboard**: Landlords can manage their listings and track enquiries.
- **Search & Filter**: Find properties by location, price range, and type.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI, TanStack Query.
- **Backend**: Express.js, Node.js.
- **Database**: PostgreSQL with Drizzle ORM.
- **Auth**: Session-based authentication.

## Getting Started
1. The app is ready to run on Replit.
2. Uses a local PostgreSQL database.
3. Database schema is managed via Drizzle.
